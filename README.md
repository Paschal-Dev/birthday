# Birthday Website

A beautiful 3D birthday website built with HTML, CSS, JavaScript, and Bootstrap.

## Features
- Glassmorphism design and glowing text effects
- Floating 3D emojis and stickers
- Interactive 3D rotating photo cube
- Dynamic tilt effect on the message card

## Setup
1. Clone this repository.
2. Place three images named `img1.jpg`, `img2.jpg`, and `img3.jpg` inside the `assets/` directory.
3. Open `index.html` in your browser.
